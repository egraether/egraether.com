award-winning HTML5 Browsergame for building marbleruns
This game was created in collaboration of art and engineering students and submitted to Mozilla Game On 2010 where it was awarded Best Overall. It was developed using PrototypeJS and box2d-js, the graphics rendering was done via Context2D of the HTML <canvas>. New features are added everytime when the length of the whole run reaches a certain number, so keep on building!
